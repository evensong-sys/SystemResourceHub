Finding tools for plural systems shouldn’t be this hard.

This project is a collection of apps, guides, and general resources for plural systems. It exists because everything is scattered, things break or disappear without warning, and people end up having to figure things out while already overwhelmed.

The goal here is simple: make that process easier and more manageable.

## What’s here

* Lists of apps and tools that are actually usable for plural systems
* Guides for things like backing up data, switching platforms, or starting over
* Alternatives and comparisons so you’re not stuck relying on one thing
* Some self-hosted and open source options for people who want more control

## Philosophy

This project is built around a few basic ideas.

Accessibility matters more than polish. If something only works when you have energy, it’s not very useful.

No gatekeeping. You don’t need specific labels or a diagnosis to use any of this.

Preparation matters. It’s better to have options ready before something breaks.

And most importantly, this should not be turned into something that exploits people. A lot of systems are vulnerable, and this kind of information should stay accessible.

## Important notes

This is not medical or therapeutic advice.

Not every tool here will work for every system.

Things will change. Apps shut down, features get removed, and services disappear. Part of this project is helping people deal with that without panic.
